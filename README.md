# Point & Click Adventure Prototype (AGS)

This repository contains a small point-and-click adventure prototype built using Adventure Game Studio (AGS).

The project demonstrates the basic mechanics required for a classic adventure game, including character movement, interaction with objects, an inventory item, background music, and a simple puzzle that concludes the demo.

## Features

- Custom animated player character
- Point-and-click movement
- Inventory system
- Object interaction (key and door)
- Background music
- End-of-demo sequence

## Gameplay

In this short prototype the player can:

1. Walk around the room
2. Pick up a key
3. Use the key on the door
4. Trigger the end of the demo

## Technologies Used

- Adventure Game Studio (AGS)
- GIMP for sprite editing
- OGG audio for music playback

## Project Structure

Audio/
Rooms/
Sprites/
GlobalScript.asc
Game.agf

## Running the Project

1. Open the project using Adventure Game Studio
2. Press F5 or click Run Game

## Notes

This project is intended as a prototype/demo and can easily be expanded with:

- Additional rooms
- Dialogue systems
- More puzzles
- Inventory combinations
